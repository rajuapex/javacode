# Introduction to Core Spring 5.x and REST

This repository contains a comprehensive suite of lab solutions for mastering **Spring Framework 5.x**, **Spring Boot 2.x**, and **RESTful Web Services**. The project is structured as a Maven Multi-Module project, allowing for centralized build management while keeping individual lab topics isolated.

## 🚀 Project Structure

The project is divided into functional modules, each representing a specific milestone in the Spring learning path:

### 1. Core Spring Framework
* **spring-javaconfig**: Mastering Dependency Injection using Java-based configuration (`@Configuration`, `@Bean`).
* **spring-bean-dependencies**: Understanding bean lifecycles and complex wiring.
* **spring-jdbc**: Working with `JdbcTemplate` for simplified database access.

### 2. Spring Boot
* **spring-boot-introduction**: Getting started with auto-configuration and the Spring Initializr.
* **spring-boot-flightservice**: A hands-on example of building a functional service with Boot.

### 3. Persistence & Data
* **spring-data-jpa**: Introduction to the Repository abstraction.
* **spring-data-query-methods**: Creating custom queries using method name conventions and `@Query`.
* **spring-data-jpa-boot-db**: Integration of H2/External databases with Spring Data.

### 4. RESTful Web Services
* **spring-rest-intro**: Building your first `@RestController`.
* **spring-rest-client**: Using `RestTemplate` or `WebClient` to consume APIs.
* **spring-rest-services**: Deep dive into status codes, headers, and error handling.
* **spring-hotel-reservation**: A complete end-to-end REST project for hotel management.
